Detailed Description

Calculates distance between activated dipoles and localized dipoles.

getDistance finds the distance between simulated and calculated dipoles. It lists the calculated Dipoles in the first two columns ordered by best pseudo-Z-score from top to bottom (see function bestPseudoZ) The last two columns of bestDistance list the distance between the calculated dipole and the simulated ones column three lists the distance of dipole in column one, column four lists the distance of dipole in column two

Parameters:
t_ForwardSolutionThis holds the Lead Field Matrix; of the type sl_CForwardSolution.
bestPseudoZHolds the 50 dipoles with best pseudo-Z-score
p_DipoleHolds information about the activated dipole pair.
Return values:
dipol_coordLists coordinates of activated dipoles
Z_coordLists coordinates of localized dipoles with best pseudo-Z-score
distancelists distances between best localized and activated dipole pair Calculates distance offset

getOffset finds the distance between activated dipole in simulation leadfeald and the closest dipole in the localization lead field. It is required that both lead fields are from the same forward solution as they have to be set in the same coordinate system

Parameters:
t_ForwardSolutionLocalizationThis holds the Lead Field Matrix used for localization; of the type sl_CForwardSolution.
p_DipoleHolds information about the activated dipole pair.
Return values:
OffsetHolds to distance values. The first value is the offset for dipole 1 and the second value is the offset for dipole 2
